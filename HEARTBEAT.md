# HEARTBEAT.md - 市场监控检查清单

每次收到心跳时，按以下优先级检查。**交易时段（9:15-15:00 工作日）优先检查前3项，非交易时段检查后2项。**

## 检查项（按优先级）

### 交易时段
1. **大盘情绪** — 用 `china-stock-sentiment` 检查今日整体市场情绪分数，有异常（<-5 或 >+7）时主动汇报
2. **热点板块** — 是否有新的政策驱动板块？新闻联播有无重要措辞？
3. **用户关注标的** — 读 `MEMORY.md` 中的关注列表，有舆情异动时提醒

### 非交易时段
4. **隔夜外盘** — 美股、港股大幅波动是否可能影响明日 A 股开盘情绪？
5. **重要公告扫描** — 上市公司公告（增持/减持/重组/业绩预告）中有无值得关注的

## 静默条件

- 深夜 23:00 - 8:00，无重大事件时不打扰
- 非交易日（周末+节假日），仅在有重大政策/外盘事件时提醒
- 情绪分数在 -2 到 +3 之间（正常范围）时，直接回 `HEARTBEAT_OK`
